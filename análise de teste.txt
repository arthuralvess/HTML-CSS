QA - Quality Assurance
(Garantia de qualidade)    -> verifica: Se o software esta de acordo com o requisito.
                                                                Se há defeitos e/ou melhorias.

Os testes são aplicados conforme o desenvolvimento do produto.
tipo de tests: Positivio : testa as funcionalidades de maneira efetiva
Negativo: testa as funcionalidades de software de maneira a forçar erros
smoke: testa as unidades simples de software, como uma construção de códigos
stress: submete o software a situações extremas, para avaliar seu comportamento.
